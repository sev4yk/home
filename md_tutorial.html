Основний синтаксис Markdown

Акценти

Ви можете додати наголос, зробивши текст жирним або курсивом.

Жирний

Щоб виділити жирний текст, додайте дві зірочки або підкреслення перед і після слова чи фрази. Щоб виділити середину слова для наголосу, додайте дві зірочки без пробілів навколо літер.

Markdown HTML Візуалізований вивід
хутін **пуйло** ла-ла-ла-ла-ла хутін <strong>пуйло</strong> ла-ла-ла-ла-ла хутін пуйло ла-ла-ла-ла-ла
хутін __пуйло__ ла-ла-ла-ла-ла хутін <strong>пуйло</strong> ла-ла-ла-ла-ла хутін пуйло ла-ла-ла-ла-ла
хутін**пуйло**ла-ла-ла-ла-ла хутін<strong>пуйло</strong>ла-ла-ла-ла-ла хутінпуйлола-ла-ла-ла-ла

Курсив

Щоб виділити курсивом текст, додайте одну зірочку або підкреслення перед і після слова чи фрази. Щоб виділити курсивом середину слова для наголосу, додайте одну зірочку без пробілів навколо літер.

Markdown HTML Візуалізований вивід
хутін *пуйло* ла-ла-ла-ла-ла хутін <em>пуйло</em> ла-ла-ла-ла-ла хутін пуйло ла-ла-ла-ла-ла
хутін _пуйло_ ла-ла-ла-ла-ла хутін <em>пуйло</em> ла-ла-ла-ла-ла хутін пуйло ла-ла-ла-ла-ла
хутін*пуйло*ла-ла-ла-ла-ла хутін<em>пуйло</em>ла-ла-ла-ла-ла хутінпуйлола-ла-ла-ла-ла

Жирний та курсив

Щоб виділити текст жирним і курсивом одночасно, додайте три зірочки або підкреслення перед і після слова чифрази.

Markdown HTML Візуалізований вивід
хутін ***пуйло*** ла-ла-ла-ла-ла хутін <strong><em>пуйло</em></strong> ла-ла-ла-ла-ла хутін пуйло ла-ла-ла-ла-ла
хутін ___пуйло___ ла-ла-ла-ла-ла хутін <strong><em>пуйло</em></strong> ла-ла-ла-ла-ла хутін пуйло ла-ла-ла-ла-ла
хутін __*пуйло*__ ла-ла-ла-ла-ла хутін <strong><em>пуйло</em></strong> ла-ла-ла-ла-ла хутін пуйло ла-ла-ла-ла-ла
хутін **_пуйло_** ла-ла-ла-ла-ла хутін <strong><em>пуйло</em></strong> ла-ла-ла-ла-ла хутін пуйло ла-ла-ла-ла-ла
хутін***пуйло***ла-ла-ла-ла-ла хутін<strong><em>пуйло</em></strong>ла-ла-ла-ла-ла хутінпуйлола-ла-ла-ла-ла

Блок -цитати

Щоб створити цитату, додайте > перед абзацом

>Не вірю у силу слова я, а вірю в силу пі*дюлей.

Виведений результат виглядає так:

Не вірю у силу слова я, а вірю в силу пі*дюлей.

Цитати з кількома абзацами

Блок - цитати можуть містити кілька абзаців. Додайте a >до порожніх рядків між абзацами.
