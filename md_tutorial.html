Ви можете додати наголос, зробивши текст жирним або курсивом.

Жирний

Щоб виділити жирний текст, додайте дві зірочки або підкреслення перед і після слова чи фрази. Щоб виділити середину слова для наголосу, додайте дві зірочки без пробілів навколо літер.

Markdown HTML Візуалізований вивід
хутін **пуйло** ла-ла-ла-ла-ла хутін <strong>пуйло</strong> ла-ла-ла-ла-ла хутін пуйло ла-ла-ла-ла-ла

Курсив

Щоб виділити курсивом текст, додайте одну зірочку або підкреслення перед і після слова чи фрази. Щоб виділити курсивом середину слова для наголосу, додайте одну зірочку без пробілів навколо літер.

Markdown HTML Візуалізований вивід
хутін *пуйло* ла-ла-ла-ла-ла хутін <em>пуйло</em> ла-ла-ла-ла-ла хутін пуйло ла-ла-ла-ла-ла

Заголовки

Щоб створити заголовок, додайте цифри ( #) перед словом або фразою. Кількість цифрових знаків, які ви використовуєте, має відповідати рівню заголовка. Наприклад, щоб створити заголовок третього рівня (<h3>), використовуйте три цифри (наприклад, ### My Header). >>>>>>> Stashed changes

Блок -цитати

Щоб створити цитату, додайте > перед абзацом

>Не вірю у силу слова я, а вірю в силу пі*дюлей.

Виведений результат виглядає так:

Не вірю у силу слова я, а вірю в силу пі*дюлей.

Цитати з кількома абзацами

Блок - цитати можуть містити кілька абзаців. Додайте a >до порожніх рядків між абзацами.
